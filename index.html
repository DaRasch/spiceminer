Welcome to the spiceminer documentation¶

The spiceminer package is a high level python wrapper around NASAs CSPICE library. It concentrates on an easy to use API for analysing the data and doesn’t include features to change or create data. If you are looking for a more fully featured wrapper, try these:

  • PySPICE
  • SpiceyPy

You can get the source code here.
